node-pywws
==========

[![Build Status](https://travis-ci.org/nicklasnygren/node-pywws.svg?branch=master)](https://travis-ci.org/nicklasnygren/node-pywws)

Express.js server for [PyWWS](https://github.com/jim-easterbrook/pywws) weather data.

Start server
```sh
npm start
```

Run tests
```sh
npm test
```
